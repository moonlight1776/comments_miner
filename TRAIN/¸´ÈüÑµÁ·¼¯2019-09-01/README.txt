1. 复赛训练集包括：
   （1）Train_makeup_reviews.csv和Train_makeup_labels.csv为14000条化妆品领域的标注数据；
   （2）Train_laptop_reviews.csv和Train_laptop_labels.csv为900条笔记本电脑领域的标注数据；
   （3）Train_laptop_corpus.csv为25000条笔记本电脑领域的未标注语料。

2. 复赛测试集为约4000条笔记本电脑领域的数据。

3. 笔记本电脑领域的category包括：
   { 硬件&性能、软件&性能、外观、使用场景、物流、服务、包装、价格、真伪、整体、其他 }

4. 最终提交格式与初赛一致；

5. 数据使用要求：
  （1）可以使用开源词向量，可以使用开源无标注数据或复赛提供的数据自行训练词向量；
  （2）可以使用开源工具进行分词、词性标注、依存句法分析；
  （3）可以使用开源的预训练语言模型(包括ELMo、GPT1.0、GPT2.0、BERT等);
  （4）禁止使用官方提供的复赛训练集以外的数据参与完成本赛题规定的任务;
  （5）禁止使用（3）以外的其他外部模型、API等对未标注的笔记本电脑语料进行标注，或参与完成本赛题规定的任务（注意：利用官方提供的复赛训练集自行训练的模型不属于外部模型，因此使用这类模型或（3）中的模型生成的标签可以使用，但不能人工参与筛选等）；
  （6）若出现以上未明确说明的数据使用问题，请在钉钉讨论群（群号：23329732）询问管理员，得到肯定答复方可使用，否则如若擅自使用，可能被取消名次。
